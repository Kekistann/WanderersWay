Installation

⚠️ Do not install this list on top of an existing Skyrim modded setup. A clean install is required.

Step 1 — Prepare Skyrim

Install a fresh copy of Skyrim Special Edition via Steam
Launch the game once to the main menu, then close it
Install all required CC content listed above

Step 2 — Install Wabbajack

Download Wabbajack from wabbajack.org
Place Wabbajack.exe in a folder such as C:\Wabbajack
Run it and let it update

Step 3 — Install Wanderer's Way

In the Wabbajack UI, browse the gallery and find Wanderer's Way, or download the .wabbajack file from this repository
Set your Installation Location — somewhere outside Program Files, e.g. C:\WanderersWay
Set your Download Location — can be the same folder or a separate drive
Click the Start button and log in with your Nexus Premium account
Let Wabbajack run — this may take a while depending on your connection

Step 4 — Post-Installation

Open Mod Organizer 2 from your installation folder
Launch the game once through MO2 using the SKSE option
Do not skip the MCM setup step below
